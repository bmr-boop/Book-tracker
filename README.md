# 📚 Book Tracker

A Python command-line app to track books 
you want to read, are reading, and have finished.

## Features
- Add books with title, author, and status
- Filter by reading status
- Rate finished books (1–5 stars)
- Delete books
- Auto-saves to a JSON file

## How to Run
python books.py

## Built With
- Python 3
